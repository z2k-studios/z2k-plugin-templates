Just starting out with adding templates to your existing Vault? One of the first things you need to decide is how to approach organizing templates within your Vault. 

Methods available are: 
1. The default "Templates" Folder located in every folder. 
2. Use a hidden ".Templates" Folder located in every folder. 
3. Use subfolders inside a root templates folder (not recommended)
4. Use template file extensions.
