# Convert to block template
Turns a file into a [[Block Templates|block template]] directly from Obsidian's file explorer.

## Where It Appears

| Location | Appears? | Condition |
| - | - | - |
| Editor context menu | No | — |
| File explorer menu | Yes | File is not already a block template |
| Command Palette | Yes | See [[Convert to Block Template]] |

> [!TIP] See Command Palette Documentation
> This is the context menu version of [[Convert to Block Template]]. See that page for full details on YAML changes, file extension behavior, and when to use this command.
