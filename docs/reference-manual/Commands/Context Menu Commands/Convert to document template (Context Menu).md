# Convert to document template
Turns a file into a [[Types of Template Files#Document Templates|document template]] directly from Obsidian's file explorer.

## Where It Appears

| Location | Appears? | Condition |
| - | - | - |
| Editor context menu | No | — |
| File explorer menu | Yes | File is not already a document template |
| Command Palette | Yes | See [[Convert to Document Template]] |

> [!TIP] See Command Palette Documentation
> This is the context menu version of [[Convert to Document Template]]. See that page for full details on YAML changes, file extension behavior, and when to use this command.
