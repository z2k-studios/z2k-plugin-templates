# Switch to .md Extension (Context Menu)
Changes a template file's extension from `.template` or `.block` to `.md` while preserving template functionality. This assumes that the template file was made while [[Template File Extensions]] were enabled.

## Where It Appears

| Location            | Appears? | Condition                                  |
| ------------------- | -------- | ------------------------------------------ |
| Editor context menu | No       | —                                          |
| File explorer menu  | Yes      | File has `.template` or `.block` extension |
| Command Palette     | Yes      | See [[Switch to .md extension]]            |

> [!TIP] See Command Palette Documentation
> This is the context menu version of the identically named Obsidian Command "[[Switch to .md extension]]". See that page for full details on behavior, examples, and related commands.
