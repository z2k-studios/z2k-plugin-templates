# Switch to .md extension
Changes a template file's extension from `.template` or `.block` to `.md` while preserving template functionality.

## Where It Appears

| Location | Appears? | Condition |
| - | - | - |
| Editor context menu | No | — |
| File explorer menu | Yes | File has `.template` or `.block` extension |
| Command Palette | Yes | See [[Convert to markdown template]] |

> [!TIP] See Command Palette Documentation
> This is the context menu version of [[Convert to markdown template]]. See that page for full details on how template status is preserved when changing the file extension.

> This command is being renamed from "Convert to markdown template" to "Switch to .md extension" – see [GitHub issue #136](https://github.com/z2k-studios/z2k-plugin-templates/issues/136).
