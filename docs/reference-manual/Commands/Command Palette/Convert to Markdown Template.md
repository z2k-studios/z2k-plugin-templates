# Command - Name

## Overview


## Detailed Description of What It Does

    - Sets the [[z2k_template_type]] YAML Property to `document-template` or `block-template` based on what type of file it was previously 
    - If the file currently has either a `.template` or `.block` file extension, it will rename it to be `.md`.
