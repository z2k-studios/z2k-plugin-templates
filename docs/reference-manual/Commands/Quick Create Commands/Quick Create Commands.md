# Quick Create Commands
Quick Create Commands let you define custom commands that appear in Obsidian's Command Palette. Each command combines a specific action with pre-configured settings, eliminating repetitive prompts for your most common template workflows.

## Contents
For more information, please see:

1. [[Quick Create Commands Overview]] — What Quick Create Commands are and why you'd use them
2. [[Managing Quick Create Commands]] — How to add, edit, delete, and reorder commands
3. [[Assigning Hotkeys to Quick Create Commands]] — Binding keyboard shortcuts to your commands
4. [[Quick Create Command Examples]] — Sample configurations for common workflows
5. [[Perform Quick Command]] — A meta-command for accessing all Quick Create Commands

> [!DANGER] IMPLEMENTATION STATUS
> Quick Create Commands are partially implemented. The current implementation supports only the "Create New File" action with a fixed target folder. The full feature set described in this documentation (action types, template selection, source text options) is planned but not yet available. See [GitHub issue #138](https://github.com/z2k-studios/z2k-plugin-templates/issues/138) for tracking.
