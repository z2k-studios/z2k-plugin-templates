# Perform Quick Command
The "Perform Quick Command..." command provides a picker interface for accessing all your configured Quick Create Commands in one place.

## How It Works
Instead of searching the Command Palette for individual Quick Create Commands or remembering their hotkeys, invoke this single command to see a list of all available Quick Create Commands.

1. Open Command Palette (`Ctrl/Cmd + P`)
2. Search for "Perform Quick Command"
3. Select "Z2K: Perform Quick Command..."
4. A picker appears showing all your Quick Create Commands
5. Select one to execute it

## When to Use This
**Use "Perform Quick Command..." when:**
- You have many Quick Create Commands and can't remember all their hotkeys
- You want to browse available commands before choosing
- You're introducing someone to your Quick Create Commands

**Use individual Quick Create Commands when:**
- You know exactly which command you want
- Speed is critical (one hotkey vs. two selections)

## Suggested Hotkey
Consider assigning `Ctrl+Shift+Q` to this command for quick access to the Quick Create picker.

## Picker Display
Commands appear in the order you've configured in settings. Each entry shows:
- The command name
- Context information (target folder or template)
