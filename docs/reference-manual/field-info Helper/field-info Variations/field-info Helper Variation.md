# field-info
The `{{field-info}}` helper function is the primary method for silently specifying [[field-info Helper|field-info]] information about a field. This variation is a [[Silent Helper Functions]], meaning that it does not output any data into the resultant file when it is finalized. 

For more information, please see [[field-info Helper]].
