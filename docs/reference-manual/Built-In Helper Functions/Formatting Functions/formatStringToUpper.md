# formatStringToUpper Helper
If you want to force the data being used for a Z2K Template Field to be rendered in all caps, use the `formatStringToUpper` Helper function:

```
{{formatStringToUpper fieldname}}
```

where:
- `formatStringToUpper` is the predefined name of the helper function for formatting text to all caps
- `fieldname` is the name of the field that will receive the data to be capitalized
