# URI Commands
Each Z2K Templates URI includes a required `cmd` parameter that determines what the plugin does. Four commands are available.

## Contents
For more information, please see:
1. [[URI Command - new]] – Create a new note from a template
2. [[URI Command - continue]] – Fill remaining fields in an existing WIP note
3. [[URI Command - insertBlock]] – Insert a block template into an existing note
4. [[URI Command - fromJson]] – Execute a full command packaged as a JSON string (advanced)
