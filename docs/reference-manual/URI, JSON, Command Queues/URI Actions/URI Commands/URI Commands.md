# URI Commands
Each Z2K Templates URI includes a required `cmd` parameter that determines what the plugin actually does. Options are listed below.

## Contents
For more information, please see:
1. [[URI Command - new]] – Create a new note from a template
2. [[URI Command - continue]] – Fill remaining fields in an existing WIP note
3. [[URI Command - upsert]] – Create a note if it doesn't exist, or update it if it does
4. [[URI Command - insertBlock]] – Insert a block template into an existing note
5. [[URI Command - fromJson]] – Execute a full command packaged as a JSON string (advanced)
