# URI, JSON, and Command Queues
Z2K Templates can be triggered externally - through URIs, JSON data packages, and queued command files. These three systems work together: [[JSON Packages]] define the data format, [[URI Actions]] use that format to trigger commands via links, and [[Command Queues]] use it to queue commands as files for batch or automated processing.

## Contents
For more information, please see:
1. [[JSON Packages]] — The standard data format for passing commands and field data into Z2K Templates
2. [[URI Actions]] — Triggering template commands from outside Obsidian via URI links
3. [[Command Queues]] — Queuing template commands as files for automated or batch processing
