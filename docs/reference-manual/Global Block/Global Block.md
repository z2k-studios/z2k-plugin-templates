# Global Block
The **Global Block** is a vault-wide block of template content – specified in the Z2K Templates Plugin's [[Global Block Settings]] page – that is automatically prepended to every template before rendering.

Unlike [[System Blocks|System Blocks]], which are hierarchical and embedded throughout the folder structure of a vault, the Global Block is a single, flat entry point: one block that applies everywhere, configured exclusively through the plugin [[Settings Page|settings]].
