# templateVersion

Like [[templateName]], the `{{templateVersion}}` built-in field will include the version number of a template into in any file created from that template, assuming it has been specified. 

Note: A template uses the [[YAML Configuration Properties|YAML entry]] `z2k_template_version` to specify its current version.
