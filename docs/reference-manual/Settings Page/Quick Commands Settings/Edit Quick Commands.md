# Edit Quick Commands
The **Edit Quick Commands** button opens the Quick Commands editor — a modal where you add, configure, reorder, and delete your Quick Commands.

## Opening the Editor
In the **Quick Commands** settings section, click **Edit Quick Commands**. The editor opens as a modal over the settings panel.

## Editor Layout
Each Quick Command appears as a card with five fields:

| Field | Description |
|-------|-------------|
| **Name** | Display name shown in the Command Palette |
| **Action** | "Create New File" or "Insert Block" |
| **Target Folder** | Destination folder path, `/` for vault root, or leave empty to prompt |
| **Template File** | Template file path, or leave empty to prompt |
| **Source Text** | None, Selection, or Clipboard |

Each card also has row controls: **↑** (move up), **↓** (move down), **+** (insert below), **×** (delete).

Use **Add Command** at the bottom of the editor to append a new card at the end of the list.

> [!NOTE] See the Quick Command Reference Pages for more details
> For full documentation on each property and example configurations, see [[Managing Quick Commands]] and [[Quick Command Examples]].

## Saving
Click **Save** to apply all changes. Click **Cancel** to discard. Changes do not take effect until you click **Save**.
