# Settings Page
The [[Z2K Templates Plugin|Z2K Templates Plugin]] is configured through Obsidian's built-in [Settings interface](https://help.obsidian.md/settings). Open it via **Settings → Community Plugins → Z2K Templates**.

## Contents
The settings panel is organized into the following sections:

1. [[General Settings]] – Core plugin settings: template root folder, creator name, embedded folder name, and file terminology
2. [[Command Queue Settings]] – Automated processing of queued JSON command files
3. [[Error Logging Settings]] – Log file path and severity level
4. [[Quick Commands Settings]] – Custom commands for quickly creating files from the Command Palette
5. [[Advanced Settings]] – File extension handling, global block content, and custom Handlebars helper functions
