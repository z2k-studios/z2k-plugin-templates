# File Extension Settings
Settings that control whether the plugin uses custom file extensions – `.template` and `.block` – to distinguish template files from regular Markdown files.

This setting turns on a whole new way for storing templates in your vault. For the full discussion of why custom file extensions exist, how they interact with Obsidian, and recommendations for adopting them, see [[Template File Extensions]].
