# Template Files Visible in File Explorer
Controls whether `.template` and `.block` files appear in Obsidian's file explorer. 

- **Default:** Off
- **Type:** Toggle

Note: This setting only appears when [[Use Template File Extensions]] is enabled.

When **off**, template files are hidden from Obsidian's file explorer, search results, and graph view. They still function as templates – the plugin can find and use them – but they stay out of your way during normal vault navigation.

When **on**, template files appear in the file explorer alongside regular files. This is useful when you need to browse, move, or rename template files directly.

> [!NOTE]
> You can also toggle visibility from the Command Palette using the [[Make .template and .block templates visible-hidden]] command, without opening Settings.
