# Custom Helper Settings
Settings for writing and registering custom Handlebars helper functions in JavaScript. This is a very advanced feature, and one to be used with caution (see Warning below). For the full discussion of helper functions and how to design them, see [[User Defined Helper Functions]].

## Contents
For more information, please see:
1. [[Enable Custom Helpers]] – Toggle to activate the custom helpers feature
2. [[Edit Custom Helpers]] – Code editor for writing and registering custom helper functions

> [!WARNING]
> Custom helpers execute **arbitrary JavaScript** with full access to your vault, files, and the Obsidian API. Only enable this if you wrote the helper code yourself or fully trust its source.
