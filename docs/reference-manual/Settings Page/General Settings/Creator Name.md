# Creator Name
The value inserted into the built-in `{{creator}}` template field whenever a new file is created. Set this to your name, handle, or alias – or anything else you want stamped globally into every new file.

- **Default:** Empty
- **Maximum length:** 100 characters

If left blank, `{{creator}}` resolves to an empty string. See the [[creator|Creator]] built-in field for more information.

If you share your templates with others, we highly encourage you to add the creator field into the YAML frontmatter.

> [!TIP] Beyond Just Your Name
> This field accepts any text, not just a personal name. A copyright notice, a company name, a website URL – anything you want inserted into every new file via `{{creator}}` works here. The reference page for the [[creator|Creator]] built-in field has a number of examples.
