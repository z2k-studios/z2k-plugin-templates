# General Settings
Z2K Templates' core plugin settings that control how Z2K Templates behave.

![[general-settings-panel.png]]
(Screenshot: The General Settings section of the Z2K Templates settings panel)

## Contents
The following general settings are available:

1. [[Templates Root Folder]] – Scopes the entire plugin to a specific folder in your vault
2. [[Creator Name]] – Sets the value inserted into the `{{creator}}` field
3. [[Templates Folder Name]] – Defines the folder name the plugin uses to identify template folders
4. [[Name for Files]] – Sets the terminology used for files throughout the plugin interface

> [!DANGER] INTERNAL NOTES
> - The settings UI currently shows no "General" heading above this group of settings. See [GitHub issue #152](https://github.com/z2k-studios/z2k-plugin-templates/issues/152).
