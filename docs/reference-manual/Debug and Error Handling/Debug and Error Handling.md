# Debug and Error Handling
Z2K Templates uses a layered approach to error handling – inline validation catches problems as you fill in fields, modal dialogs surface errors during template processing, and a configurable error log captures everything behind the scenes. This section covers how these layers work together and how to use them when something goes wrong.
