# Installing via Obsidian Community Plugins

> [!NOTE]
> Z2K Templates is not yet available in the Obsidian Community Plugin browser. This page will become active once the plugin has completed Obsidian's review process. In the meantime, see [[Installing via BRAT]] to install the current beta release.

Z2K Templates is distributed through Obsidian's built-in community plugin browser — no external downloads or manual file placement required.

## Prerequisites
Community plugins must be enabled in your vault. If you see a **Restricted Mode** toggle in **Settings → Community plugins**, turn it off to allow third-party plugins.

## Installation Steps
1. Open **Settings → Community plugins**
2. Click **Browse**
3. Search for **Z2K - Templates**
4. Click **Install**
5. Once installation completes, click **Enable** — or return to the installed plugins list and toggle Z2K - Templates on

Z2K Templates is now active in your vault. Head to [[Introduction]] to get started.
